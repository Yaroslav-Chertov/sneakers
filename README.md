# Интернет-магазин кроссовок

> SPA интернет-магазин с каталогом товаров, корзиной и избранным. Vue 3, TailwindCSS, Axios, анимации. Поиск, фильтрация и оформление заказа.

**Продакшн:** https://sneakers-crw5my150-yaroslavs-projects-c6eb330e.vercel.app/

---

## О проекте

Интернет-магазин кроссовок — одностраничное приложение с возможностью просмотра каталога товаров, добавления в корзину и избранное, а также оформления заказа.

Проект ориентирован на создание плавного пользовательского опыта за счёт анимаций, реактивного состояния и удобной навигации.

---

## Стек

| Слой | Технология |
|---|---|
| UI | Vue 3 |
| Роутинг | Vue Router |
| Стили | TailwindCSS |
| HTTP | Axios |
| Анимации | auto-animate |
| Архитектура | Composition API / Options API |
| Линтинг | ESLint, Prettier |

---

## Архитектура

```
src/
├── components/        # UI-компоненты (карточки, корзина, хедер)
├── pages/             # Страницы приложения
├── router/            # Конфигурация маршрутов
├── composables/       # Переиспользуемая логика
├── assets/            # Статические ресурсы
├── App.vue            # Корневой компонент
└── main.js            # Точка входа
```

---

## Ключевые решения

- Реактивное состояние через Composition API
- Разделение логики и UI через composables
- Анимации интерфейса через auto-animate
- Работа с API через Axios
- Переиспользуемые компоненты каталога

---

## Ключевые модули

**Каталог товаров** — отображение списка кроссовок с поиском и фильтрацией.

**Корзина** — добавление и удаление товаров, подсчёт общей стоимости.

**Избранное** — сохранение понравившихся товаров.

**Оформление заказа** — отправка данных заказа через API.

---

## Взаимодействие с API

- Используется тестовый API Mokky
- Получение списка товаров
- Отправка заказов
- Axios для HTTP-запросов

Документация: https://mokky.gitbook.io/welcome
